# 什么是HTML？

HTML 指的是超文本标记语言 (Hyper Text Markup Language)。

# HTML的基本结构

声明
<! DOCTYPE html>
网页开始
<html>
           头部标签
         <head> 
                  解决中文问题的字符集编码
                <meta charset = "UTF-8">
                  网页标题
                <title></title>
         </head>
          网页主体
         <body>
              各种标签
          （（内容写在此处！！！！））
         </body>
</html>

# 引用术语标签（了解）:

# 【abbr】定义缩写

    <abbr title="">（被缩写的内容）</abbr>

title:包含了英文短语的具体内容。

提示

对缩写进行标记能够为浏览器、翻译系统以及搜索引擎提供有用的信息。

当鼠标放在（被缩写的内容）时，会提示title里面的内容。

# 【address】定义地址

    <address></address>

提示

定义文档或文章的作者/拥有者的联系信息。它会斜体显示。

# 【bdo】定义文字方向（定义双流向覆盖）

    <bdo dir=""></bdo>

dir: 定义了文字的方向

【 属 性 值：ltr (默认从左到右） rtl 从右到左】

提示

用于覆盖当前文本方向。

# 【blockquote】定义长的引用（块引用）

    <blockquote></blockquote>

提示

所有文本都会从常规文本中分离出来，经常会在左、右两边进行缩进（增加外边距），而且有时会使用斜体。也就是说，块引用拥有它们自己的空间。

# 【q】短的引用语

    <q></q>

提示

浏览器经常在引用的内容周围添加引号。

# 【cite】引用、引证

    <cite></cite>

提示

表示它所包含的文本对某个参考文献的引用，比如书籍或者杂志的标题。引用的文本将以斜体显示。

注意

隐藏的功能：它可以使你或者其他人从文档中自动摘录参考书目。

# 【dfn】定义一个定义项目（尽量少用）

    <dfn></dfn>

提示

可标记那些对特殊术语或短语的定义。还可能有助于创建文档的索引或术语表。

注意

它会把内容变倾斜，可以使用<abbr>代替。
